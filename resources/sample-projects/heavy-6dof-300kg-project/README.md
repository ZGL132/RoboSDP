# Heavy 6DOF 300kg Test Project

This sample project defines a floor-mounted 6-axis industrial robot for RoboSDP main-chain testing.

- Rated payload: 300 kg
- Peak payload: 330 kg
- Nominal reach: 2.8 m
- Robot type: 6R serial manipulator with spherical wrist
- Intended tests: requirements, topology, DH kinematics, FK/IK preview, workspace cache, dynamics, drive selection, planning scene, snapshot and export loading

The data is synthetic but sized to a typical heavy-payload handling robot.
